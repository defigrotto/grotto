A Decentralised Lotto Platform

What is Grotto?

Defi Grotto is a lotto (lottery) game that is decentralised. This means anyone can participate in the lotto game, it also means anyone can create a lotto game and earn rewards like a house. Grotto currently uses a Winner-Takes-All engine. This means for every lotto pool created, the winner takes all the winning.

Why Grotto?

One of the major use case for Blockchain is trust. Today, when you play the lottery, you are trusting a set of people to be fair and honest in choosing the winner(s). You don't have to. Blockchain Rules (Smart Contracts) could be used to manage the lotto pool, such that every player is guaranteed a fair chance.

Staking

GROTTO owners (except Governors) can stake their GROTTO to get a percentage of house cuts. The percentage is shared among all stakers. The higher the stake, the higher the rewards. Governors can determine how much from the house cuts goes to stakers. Governors can not participate in staking.

Governance

Grotto uses a few internal variables. These variables can be changed through an easy to use governance interface. A set of 11 initial Governors are hard-coded into the contract. A further 10 governors can be added to make 21 governors. Governors must have 100,000 GROTTO tokens. This value (100K) can also be changed by Governors. Anyone can be a governor if nominated by another governor.

Tokenomics

There will be an unlimited number of GROTTO tokens because tokens are minted whenever a winner is found in any pool. This is the only way new tokens can enter the system. Practically, the amount of tokens grows as more winners emerge from pools. The initial 11 governors are given 100K GROTTO each. They must hold these tokens to be able to participate in governance.

Tokenomics

The main use case for GROTTO tokens is to participate in Governance, Staking and finally Exchange for other tokens/ cryptos. In the future, GROTTO will be listed on exchanges and can be traded. GROTTO, since it's an ERC20 token can also be used to participate in other Defi projects like staking and lending on supported platforms.
